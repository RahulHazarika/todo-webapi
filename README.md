# Todo List API

Built with .NET 8 Web API

## Run the project
1. Clone the repo
2. Open in Visual Studio
3. Press F5 or run:
dotnet run

API runs on http://localhost:5021

## Endpoints
- GET /api/ToDo — get all todos
- POST /api/ToDo — add a todo
- PATCH /api/ToDo/{id} — update a todo
- DELETE /api/ToDo/{id} — delete a todo
